# Weather Forecast Application

A simple **Weather Forecast Application** built using **JavaScript, HTML, CSS, and Bootstrap**.  
This project demonstrates frontend skills like DOM manipulation, dynamic updates, and interactive UI design.

## Features
- Search for a city to view simulated weather conditions.
- Dynamic UI updates to display weather information.
- User-friendly interface with weather icons and responsive design.

## Technologies Used
- JavaScript
- HTML5 & CSS3
- Bootstrap 5
